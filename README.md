# Netflix-clone
Clone feito com #7daysofcode
Esse projeto mescla HTML e CSS em meu código a fim de criar uma cópia similar a página principal da netflix. 

No primeiro dia, foi criado o banner principal com o filme de destaque da página com auxílio de pesquisas e IAs. Porém, mesmo assim, foi um desafio pois voltei recentemente para o mundo da programação após minha rotina de ensino médio cansativa. 

No segundo dia, foi criado a parte do menu de navegação e essa foi mais simples, porém exigiu a leitura de muitos arquivos para lembrar os conceitos de flexbox e, assim, pude reaprender de uma forma mais dinâmica e entendível com a Alura. Para os modelos têm sido utilizado a ferramenta FIGMA. 

No terceiro dia, foi criado o trilho de "Minha lista" contendo filmes e séries, sendo cada um, um item. Nesse dia, foram aplicados conceitos de layout com grid, e foi relativamente simples. 

No quarto dia, fiz a lista dos filmes e séries em alta, e os conceitos foram os mesmos, mas com hover, apesar de já ter usado hover no dia anterior.

No quinto dia, mais um trilho foi feito com uma lista de 16 séries, agora aplicando o conceito de nth-of-type com hover, para assim criar um rolamento manual. Entretanto as setas não são funcionais, apenas decorativas, já que não apliquei conceitos script.
